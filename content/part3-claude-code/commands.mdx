## מצבי עבודה — `Shift+Tab` למעבר ביניהם

Claude Code תומך בשלושה מצבים. לחצו `Shift+Tab` כדי לעבור ביניהם:

| מצב | מה עושה | מתי |
|-----|---------|-----|
| **Normal** | Claude מתכנן ומבצע | עבודה יומית |
| **Plan Mode** | Claude מתכנן בלבד, לא מבצע | משימות גדולות — תכננו לפני |
| **Accept Edits** | אתם מאשרים כל שינוי ידנית | קוד רגיש, סביבת production |

## קלט עשיר — `@` ותמונות

### הפניה לקבצים עם `@`

הקלידו `@` ואחריו שם קובץ כדי להפנות את Claude ישירות לקובץ. Claude יקרא את הקובץ ויעבוד איתו — בלי שתצטרכו להעתיק-הדביק תוכן.

דוגמאות:
- `@src/auth.ts` — הפניה לקובץ ספציפי
- `@package.json` — הפניה לקובץ קונפיגורציה
- `@https://example.com/api` — הפניה ל-URL

### הדבקת תמונות

השתמשו ב-`Alt+V` (Windows/Linux) או `Option+V` (Mac) כדי להדביק תמונה מה-clipboard ישירות לתוך הפרומפט.
