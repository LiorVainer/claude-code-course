## כלל 90/9/1 — מתי להשתמש במה

**90% מהזמן — Sonnet**

Features, באגים, refactoring, code review, כתיבת טסטים. זה הסוס העבודה שלכם. מהיר, חכם, מחיר סביר.

**9% מהזמן — Opus**

ארכיטקטורה, race conditions, בעיות שדורשות חשיבה עמוקה ומרובת שלבים, security review. כשהבעיה מורכבת ושווה להשקיע.

**1% מהזמן — Haiku**

שינוי שם משתנה, תיקון import חסר, עדכון תיעוד פשוט. משימות מכניות שלא דורשות חשיבה.
