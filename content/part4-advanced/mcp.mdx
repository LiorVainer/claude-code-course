# MCP — Model Context Protocol

<Callout emoji="💡">
MCP הוא ה-API של עולם ה-AI לכלים חיצוניים — פרוטוקול סטנדרטי שמאפשר לכל מודל AI להתחבר לכל כלי חיצוני. כמו ש-REST API מחבר שרתים לעולם, MCP מחבר AI לכלים.
</Callout>

## מה זה MCP?

**Model Context Protocol** הוא תקן פתוח שפיתחה Anthropic. הוא מגדיר כיצד AI יכול לתקשר עם כלים חיצוניים — databases, APIs, ממשקי פיתוח, ועוד.

| | ללא MCP | עם MCP |
|---|---|---|
| אינטגרציה | כל מודל + כל כלי = integration מיוחד | כל מודל + כל כלי = חיבור אחיד |

## שרתי MCP נפוצים

| שרת | שימוש |
|-----|--------|
| [context7](https://context7.com) | דוקומנטציה מעודכנת של ספריות |

| [Playwright MCP](https://github.com/nicobailon/playwright-mcp) | E2E testing ואוטומציית דפדפן |
| [Elasticsearch MCP](https://github.com/elastic/mcp-server-elasticsearch) | חיפוש ושאילתות על Elasticsearch |
| [Notion MCP](https://developers.notion.com/guides/mcp/get-started-with-mcp) | ניהול דפים, databases ותוכן ב-Notion |

- [Docker Hub MCP](https://hub.docker.com/mcp) — שרתים רשמיים ומאושרים על ידי Docker
- [mcpservers.org](https://mcpservers.org/en) — קטלוג קהילתי מקיף של שרתי MCP

## שני סוגים של MCP servers

| סוג | איך עובד | דוגמאות | מתי |
|-----|----------|---------|-----|
| **Remote (HTTP)** | שרת מרוחק באינטרנט | context7, GitHub, Sentry | רוב הזמן — התקנה מהירה, תחזוקה אפס |
| **Local (stdio)** | רץ על המחשב שלכם | כלי פנימי, database מקומי | כשצריך גישה למשאבים מקומיים |

## התקנת MCP server

**Remote** — רוב ה-MCP servers הפופולריים הם remote. מוסיפים בפקודה אחת:

<Terminal title="mcp — התקנת remote server" lines={[
  { prompt: '❯', command: 'claude mcp add --transport http context7 https://mcp.context7.com/mcp' },
  { output: '✓ context7 added as remote MCP server', color: '#28c840' },
]} />

**Local** — לכלים שרצים על המחשב שלכם:

<Terminal title="mcp — התקנת local server" lines={[
  { prompt: '❯', command: 'claude mcp add --transport stdio my-db-tool npx my-db-mcp start' },
  { output: '✓ my-db-tool added as local MCP server', color: '#28c840' },
]} />

## Figma MCP — Design to Code

<Callout emoji="💡">
עם Figma MCP מחובר Claude יכול לקרוא ישירות את הקומפוננטות מ-Figma ולהמיר אותן לקוד React. אין צורך לצלם screenshot ולהסביר — Claude רואה את ה-design tokens, מרווחים, וצבעים ישירות.
</Callout>

### תהליך עבודה עם Figma MCP

1. מחברים Figma MCP
2. מדביקים קישור לפריים ב-Figma
3. Claude קורא את העיצוב ישירות
4. Claude מייצר קוד React מדויק

### התקנת Figma MCP

<Terminal title="mcp — התקנת Figma MCP" lines={[
  { prompt: '❯', command: 'claude mcp add --transport http figma-remote-mcp https://mcp.figma.com/mcp' },
  { output: '✓ Figma MCP server added to Claude Code', color: '#28c840' },
]} />
