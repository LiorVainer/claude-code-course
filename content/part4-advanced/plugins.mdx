# Plugins — תוספים מהקהילה

<Callout emoji="💡">
Plugin הוא חבילה שמכילה **אחד או יותר** מהרכיבים הבאים: Skills, Hooks, MCP servers, Agents ו-Commands.
יש plugins שמכילים רק Skill בודד, ויש כאלה שמשלבים כמה רכיבים יחד. במקום להגדיר כל דבר בנפרד — מתקינים plugin אחד.
</Callout>

## מה יכול להיות ב-Plugin?

Plugin מכיל **לפחות רכיב אחד** מהסוגים הבאים (לא חובה את כולם):

| רכיב | תפקיד |
|---|---|
| Skills | פרומפטים לשימוש חוזר |
| Hooks | אוטומציות דטרמיניסטיות |
| MCP Servers | חיבור לכלים חיצוניים |
| Agents | סוכנים מותאמים אישית |
| Commands | פקודות slash מותאמות |

## Plugins מומלצים

- **Code Intelligence Plugins** — plugins עבור שפות typed כמו TypeScript ו-Python, שמוסיפים navigation לסמלים ואיתור שגיאות אוטומטי אחרי עריכה.
- **context-mode** — plugin שחוסך עד 98% בtokens על ידי ניהול חכם של נתונים גדולים מחוץ ל-context.
