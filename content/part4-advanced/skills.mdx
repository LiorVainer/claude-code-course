# Skills — תבניות פרומפט לשימוש חוזר

<Callout emoji="💡">
Skills הם תבניות פרומפט שנטענות רק כאשר צריך אותן — **lazy loading**. הם חוסכים tokens כי הם לא טעונים בזיכרון כל הזמן, אלא רק כשמשתמשים בהם.
</Callout>

## מה זה Skill?

Skill הוא קובץ Markdown שמכיל הוראות, תבניות, ופרומפטים לשימוש חוזר. במקום לכתוב את אותו הפרומפט שוב ושוב, אתם מגדירים Skill פעם אחת ומשתמשים בו בפקודה פשוטה.

## Lazy Loading — חיסכון ב-tokens

| | ללא Skills | עם Skills |
|---|---|---|
| פרומפטים | בזיכרון תמיד → בזבוז tokens | נטענים רק כשקוראים להם → חיסכון משמעותי |

## מדריך skills.sh — קהילה

[skills.sh](https://skills.sh) הוא הרפוזיטורי הקהילתי ל-Skills. תוכלו:
- לגלות Skills שאחרים כתבו
- להוריד Skills שימושיות
- לפרסם Skills שכתבתם

## יתרונות Skills

| יתרון | תיאור |
|-------|--------|
| חיסכון ב-tokens | לא נטענים כשלא צריך אותם |
| עקביות | אותו פרומפט בכל ריצה |
| שיתוף צוות | כל חבר צוות משתמש באותן הגדרות |
| גרסאות | ניהול גרסאות עם git |
