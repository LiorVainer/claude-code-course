# simplify/ — סקירת קוד אוטומטית

<Callout emoji="💡">
`/simplify` מרים 3 sub-agents במקביל שמסקרים את הקוד שלכם מזוויות שונות. כל agent עובד בזיכרון נפרד — כמו לקבל 3 code reviews מקצועיים בו-זמנית.
</Callout>

## שלושת ה-Agents

| Agent | תחום | מה הוא בודק |
|-------|------|-------------|
| Agent 1: איכות | Quality | Bugs, error handling, edge cases, input validation |
| Agent 2: שימוש חוזר | Reusability | Duplicate code, functions שניתנות לשיתוף, abstractions |
| Agent 3: יעילות | Efficiency | Performance bottlenecks, unnecessary re-renders, N+1 queries |

## זיכרון נפרד — יתרון מרכזי

כל agent מסתכל על הקוד בעיניים טריות, ללא הנחות שנוצרו בשיחה הקודמת. זה מחקה את הדינמיקה של code review אמיתי שבו מפתח חיצוני מסתכל על הקוד.

<Callout emoji="💡">
טיפ: הריצו `/simplify` לפני כל PR. זה כמו לקחת 3 code reviewers מקצועיים לכל commit — בחינם ובלי לחכות.
</Callout>
