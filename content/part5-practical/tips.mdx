## Remote Control — שליטה מהטלפון

### מה זה?

Remote Control מאפשר לכם לנהל session של Claude Code מהטלפון בזמן שהקוד רץ על המחשב שלכם.

הפקודה `/rc` (או `/remote-control`) מציגה QR code. סרקו עם הטלפון וקיבלתם שליטה מלאה על ה-session — כולל שליחת פרומפטים, אישור פעולות, וצפייה בפלט.

<Callout emoji="💡">
הקוד נשאר על המחשב שלכם — הטלפון רק שולח פקודות. אין העברת קוד לענן.
</Callout>

## Voice Mode — דברו במקום להקליד

<Callout emoji="🎤">
Voice Mode מתגלגל בהדרגה למשתמשים — כרגע זמין לכ-5% מהמשתמשים, עם השקה רחבה בשבועות הקרובים.
</Callout>

Anthropic הוסיפו **מצב קולי** ל-Claude Code — במקום להקליד פרומפטים, אפשר פשוט לדבר. הקלידו `/voice` כדי להפעיל, דברו את הפקודה שלכם, ו-Claude יבצע.

## כלי CLI שClaude יודע להשתמש בהם

Claude Code יודע להשתמש **בכל כלי CLI מוכר** שמותקן על המחשב שלכם — מהרגע שהוא מזהה אותו, הוא משתמש בו אוטומטית. זו הדרך הכי יעילה מבחינת context לתקשר עם שירותים חיצוניים.

| כלי | שירות | דוגמה |
|------|--------|-------|
| `gh` | GitHub | PRs, issues, code review |
| `aws` | AWS | S3, Lambda, תשתיות |
| `gcloud` | GCP | deploy, שירותים |
| `sentry-cli` | Sentry | שגיאות, monitoring |
| `docker` | Docker | containers, images |
| `kubectl` | Kubernetes | pods, deployments |
