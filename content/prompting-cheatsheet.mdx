# Cheatsheet —  טיפים ותזכורות

| טיפ | תיאור | היכן בקורס |
|-----|--------|-----------|
| Context מראש | תארו את הסיטואציה לפני הבקשה | [הרגלי עבודה](/part5-practical/daily-habits) |
| Think thoroughly | "חשוב לעומק" > "step-by-step" | [הרגלי עבודה](/part5-practical/daily-habits) |
| אמרו מה לעשות | במקום "אל תעשה X" אמרו "עשה Y" | [הרגלי עבודה](/part5-practical/daily-habits) |
| `CLAUDE.md` | הגדרת conventions וpatterns | [חלק ג׳](/part3-claude-code) |
| `/plan` לפני קוד | בקשו תוכנית לפני מימוש | [היום הראשון](/part5-practical/first-day) |
| `/compact` ב-60% | נקו context לפני שהוא מלא מדי | [הרגלי עבודה](/part5-practical/daily-habits) |
| `context7` לספריות | תמיד עם דוקומנטציה מעודכנת | [context7](/part4-advanced/context7) |
| `Hooks` לגבולות קשיחים | אל תסמכו על AI לאבטחה | [Hooks](/part4-advanced/hooks) |
| `/simplify` לפני PR | סקירת קוד אוטומטית ב-3 agents | [`/simplify`](/part5-practical/simplify) |
| `verify` פקודות אימות | שימו test/lint/tsc ב-CLAUDE.md | [claude-md](/part3-claude-code/claude-md) |
| `@` הפניה לקבצים | `@src/auth.ts` במקום "תסתכל על הקובץ" | [commands](/part3-claude-code/commands) |
| `Alt+V` תמונות | הדביקו screenshots ישירות לPrompt | [commands](/part3-claude-code/commands) |
| ראיון הפוך | "מה השאלות שלך לפני שמתחילים?" | [הרגלי עבודה](/part5-practical/daily-habits) |
| `/clear` בין משימות | איפוס מלא כשמחליפים נושא | [סשנים](/part3-claude-code/stateless) |
| כלי CLI | התקינו gh, aws, gcloud | [טיפים](/part5-practical/tips) |

לתיעוד הרשמי של Anthropic על prompting: [Prompt Engineering Guide](https://docs.anthropic.com/en/docs/build-with-claude/prompt-engineering/overview)
