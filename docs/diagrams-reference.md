## File Location

All SVGs go in: `public/images/diagrams/`

## Export Settings

- Background: **OFF** (transparent)
- Dark mode: OFF for light version, ON for dark version
- Format: **SVG**
- Scale: 1x

## DiagramImage Component

The `DiagramImage` component auto-swaps light/dark:
- Light mode shows: `src` (e.g. `mcp-api.svg`)
- Dark mode shows: `src` with `-dark` inserted (e.g. `mcp-api-dark.svg`)
