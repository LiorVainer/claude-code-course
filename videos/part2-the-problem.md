# חלק ב׳ — הבעיה

> עכשיו שאתם מבינים Tokens ו-Context — בואו נראה למה רוב המפתחים עדיין עובדים לא נכון. זה לא דעה — זה ארכיטקטורה.

---

## לולאת ה-Copy-Paste

### הכירו את הלולאה

```
באג 🐛 → ChatGPT 🌐 → העתק 📋 → הדבק 📝 → תקן imports 🔧 → נשבר 💥 → חזור 🔄
```

### למה זה נשבר? ניתוח טכני

| בעיה | הסיבה הטכנית |
|------|-------------|
| אפס הקשר | ChatGPT רואה רק מה שהדבקתם |
| נמחק בין Sessions | כל שיחה מתחילה עם Context ריק |
| בזבוז Tokens | מדביקים אותם utils שוב ושוב |
| תשובות גנריות | בלי כללי פרויקט — אפס naming conventions |
| אפס יכולת ביצוע | לא יכול לקרוא קבצים או להריץ טסטים |
| APIs שלא קיימים | training cutoff → APIs שהוסרו |

### אתם הופכים ל-Human Compiler

כל תגובה דורשת מכם: להעתיק, להדביק, לתקן הזחה, לתקן imports, לטפל בקוד דבק. אתם לא חושבים — אתם מקמפלים.

### והמחיר מצטבר

כל מחזור copy-paste לוקח 2–5 דקות. מפתח טיפוסי חוזר על זה 10–15 פעם ביום. תעשו את החשבון:

| מדד | ערך |
|-----|------|
| לכל מחזור | 2–5 min |
| מחזורים ביום | 10–15× |
| סה״כ בשבוע | 3+ שעות |

## הגישה החדשה

> אחרי שהבנו למה copy-paste לא עובד — בואו נכיר מהי האלטרנטיבה

### Vibe Coding

פרדיגמת פיתוח שבה המפתח עובר מ"אדם שמקליד קוד" ל"ארכיטקט שמפקח על סוכן AI". במקום לכתוב כל שורה — מתארים כוונה, בודקים תוכניות, מאשרים שינויים.

המונח נטבע ע"י Andrej Karpathy (מייסד שותף ב-OpenAI) בתחילת 2025.

### הספקטרום — איפה אתם עכשיו?

| רמה | כלי | התפקיד שלכם |
|-----|------|-------------|
| Copy-Paste | ChatGPT | מקלידן + human compiler |
| **Agentic** | **Claude Code** | **ארכיטקט ומפקח** |

### ארבעה עקרונות

- Context הוא המלך — ה-AI צריך לראות את כל הפרויקט, לא snippet
- מפתח כארכיטקט — אתם מגדירים דרישות ומכוונים, ה-AI מבצע
- שיפור איטרטיבי — שיחה, לא prompt בודד
- סמכו אבל תוודאו — ה-AI מדייק 80–90%. תפקידכם: לתפוס את ה-10–20%

### מה משתנה בפועל

| לפני (Copy-Paste) | אחרי (Vibe Coding) |
|--------------------|---------------------|
| כותבים כל שורה | מתארים את הכוונה |
| מחפשים ב-StackOverflow | AI מחפש בקוד שלכם + דוקומנטציה מעודכנת |
| מריצים טסטים ידנית | AI מריץ ומפרש תוצאות |
| **אתם המבצעים** | **אתם הארכיטקט** |
